1) First run the server.py file. Now server runs on local host.
2) Now run the client.py file. A connection is established between server and client.
3) Now enter the desired command and press enter.
4) To perform testing run testing.py file which consists of 13 test cases. These test cases were considered to test the application.
5) To logout (or) close the application use quit command (or) enter exit.

Format of Commands:
1) register <username> <password>
2) login <username> <password>
3) write_file <filename> <content> (or) write_file <filename>
4) create_folder <foldername>
5) change_folder <foldername>
6) read_file <filename>
7) list
8) quit
